# Tkinter Calculator

A simple **GUI calculator** built using Python's `Tkinter` library. This calculator can perform basic arithmetic operations including addition, subtraction, multiplication, and division.

---

## Features

* Add, subtract, multiply, and divide two numbers
* Clear the current input
* Easy-to-use GUI with buttons for digits `0-9` and operations `+`, `-`, `*`, `/`
* Display shows the current input and results

---

---

## Usage

Run the calculator using Python:

```bash
python calculator.py
```

* Click the **number buttons** to input numbers.
* Click the **operator buttons** (`+`, `-`, `*`, `/`) to perform operations.
* Click **C** to clear the display.
* Click **=** to calculate the result.
